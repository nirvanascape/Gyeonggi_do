청소년 회원가입 대리인 회원가입

  • 청소년 본인의 인증서로 회원가입

    청소년이 가입을 하시는 경우에 청소년을 선택해 주십시오.

    청소년 선택 후 클릭하시면 회원가입이 시작됩니다.

    모든 약관이 필수동의가 필요하므로 모든 이용약관에 동의해 주십시오.

    모든 이용약관에 동의한 후 다음 단계로 이동합니다.

    신청하시는 청소년의 이름과 주민등록번호 13자리를 입력하세요.

    휴대폰 인증을 할 수 있는 본인 또는 대리인을 선택 후 휴대폰 번호를 입력하여 인증번호를 발송하세요.

    입력한 휴대폰으로 전송된 인증번호 6자리를 입력 후 인증을 완료해 주세요.

    주민등록상 거주하고 있는 지역을 선택하세요.

    간편인증 및 공동·금융 인증서 중 한 가지를 선택한 후 인증해주세요.
    ※ 금융 인증은 모바일에서만 제공합니다.

    간편 인증 방법

    간편인증을 위해, 간편인증 사업자 별로 제공하는 인증서 발급절차에 따라 인증서를 발급 하셔야 합니다.

    • 카카오 : 카카오톡 앱 > 지갑 만들기
    • 통신사패스 : PASS 앱 > 인증서
    • 삼성패스 : 삼성PASS 앱 > 인증서관리
    • KB모바일인증서 : KB스타뱅킹 > 인증센터 > KB모바일인증서 > 인증서발급/재발급
    • 페이코 : PAYCO 앱 > 더보기 > PAYCO인증서

    1. 위에 표시된 보안문자를 입력해 주세요.

    2. 인증하시는 분의 간편인증 업체를 선택한 후, 서비스 이용 약관에 동의하고 인증요청 버튼을 클릭하세요.

    3. 입력한 휴대폰 번호로 ‘정부24에서 인증요청이 왔습니다‘ PUSH 메시지가 발송됩니다. 간편인증 업체에 따라 비밀번호, 패턴, 생체 인식 등을 통해 본인인증을 완료한 후, ‘인증완료’ 버튼을 클릭해주세요.

    4. 정부24에서 주민등록등본 자동 조회를 시작합니다.

    5. 경기도의 해당 시·군의 거주확인이 완료되었습니다.

    공동·금융 인증 방법
    [공동 인증 방법]

    1. 위에 표시된 보안문자를 입력해 주세요.

    2. 인증하는 분의 공동인증서를 선택해 주세요.

    3. 인증서의 암호를 입력해 주세요.

    4. 정부 24에서 주민등록등본 자동 조회를 시작합니다.

    5. 경기도의 해당 시·군의 거주확인이 완료되었습니다.

    [금융 인증 방법]

    ※ 금융 인증은 모바일에서만 가능합니다.

    1. '정보 조회하기'를 터치하고, 인증서 제출을 '금융인증서'로 선택해주세요.

    2. 위에 표시된 보안문자를 입력하고, '다음'을 터치하세요.

    3. 인증하시는 분의 이름, 휴대폰번호, 생년월일을 입력한 후 확인 버튼을 터치하세요.

    4. 입력한 휴대폰 번호로 전송된 문자를 확인하시고, 확인코드 숫자를 입력해주세요.

    5. 인증하시는 분의 금융인증서를 선택해주세요.

    6. 금융인증서 비밀번호 6자리를 입력해주세요.

    7. 경기도의 해당 시·군의 거주확인이 완료되었습니다.

    인증이 완료된 후 다음 단계로 이동합니다.

    가입을 원하시는 아이디로 입력하고, 중복확인을 해주세요.

    영문자와 숫자, 특수문자 1개 이상을 사용한 비밀번호를 입력해 주세요.

    모든 정보 입력이 완료된 후 가입을 완료해 주세요.

    교통카드 등록 팝업에서 교통카드를 등록해주세요.

    교통카드 등록 방법
    [선불 교통카드]

    1. 교통카드 종류와 카드사를 선택한 후, 교통카드 번호를 입력해주세요.
    (※ 카드사 별로 교통카드 번호 확인 방법이 다르니, ‘번호확인 안내‘ 팝업 내용을 확인해주세요.)

    2. 교통카드가 등록되면, 지역화폐를 등록해주세요.

    [후불 교통카드]

    1.교통카드 종류와 카드사를 선택한 후, 교통카드 번호를 입력해주세요.
    (※ 카드사 별로 교통카드 번호 확인 방법이 다르니, ‘번호확인 안내‘ 팝업 내용을 확인해주세요.)

    2. ‘교통카드 등록＇을 클릭 후, 카드 인증을 완료해주세요. (※ 카카오뱅크, 씨티카드, 가족카드는 인증 절차가 없습니다.)

    3. 교통카드가 등록되면, 지역화폐를 등록해주세요.

    지역화폐를 등록하기 위해 교통카드 등록 페이지로 이동합니다.

    지역화폐 등록 방법
    [본인 지역화폐 등록]

    1. 휴대폰으로 인증 번호를 발송하고, 인증번호 6자리를 입력 후 인증을 완료해주세요.

    2. 본인의 지역화폐 번호를 입력해주세요.
    (※ 성남시, 시흥시, 김포시는 지역화폐 번호 입력을 하지 않습니다.)

    3. 지역화폐를 등록 후, 지원금을 신청해주세요.

    [본인 명의가 아닌 지역화폐 등록]

    1. 대리인의 휴대폰으로 인증 번호를 발송하고, 인증번호 6자리를 입력 후 인증을 완료해주세요.

    2. 대리인의 지역화폐 번호를 입력해주세요.
    (※ 성남시, 시흥시, 김포시는 지역화폐 번호 입력을 하지 않습니다.)

    3. 지역화폐를 등록 후, 지원금을 신청해주세요.

    교통카드와 지역화폐를 등록 후, 지원금 신청하기 버튼을 클릭해주세요.

    지원금 신청을 위해 지원금 신청 페이지로 이동합니다.
    (※ 다음에 신청하시는 경우, 마이페이지 > 지원금 신청에서 지원금을 신청하실 수 있습니다.)

    입력한 휴대폰으로 전송된 인증번호 6자리를 입력 후 인증을 완료해 주세요.

    ‘자녀추가’를 클릭해 주세요. (※ 자녀추가를 하셔야 회원가입이 가능합니다.)

    추가하려는 자녀의 이름과 주민등록번호 13자리를 입력하세요.

    자녀가 휴대폰이 있고 인증이 가능한 경우에 입력을 한 후 인증번호를 발송해 주세요. (※ 자녀의 휴대폰이 없는 경우에는 입력하지 않아도 됩니다.) 자녀가 한 명 이상인 경우, 인증을 완료하고 '자녀추가'를 클릭하여 다른 자녀의 정보를 입력하세요.

    주민등록상 거주하고 있는 지역을 선택해주세요.(※ 청소년과 대리인은 주민등록상 동일한 지역에 거주해야 합니다.)

    간편인증 및 공동·금융 인증서 중 한 가지를 선택한 후 인증해주세요. ※ 금융 인증은 모바일에서만 제공합니다.

    간편 인증 방법

    간편인증을 위해, 간편인증 사업자 별로 제공하는 인증서 발급절차에 따라 인증서를 발급 하셔야 합니다.

    • 카카오 : 카카오톡 앱 > 지갑 만들기
    • 통신사패스 : PASS 앱 > 인증서
    • 삼성패스 : 삼성PASS 앱 > 인증서관리
    • KB모바일인증서 : KB스타뱅킹 > 인증센터 > KB모바일인증서 > 인증서발급/재발급
    • 페이코 : PAYCO 앱 > 더보기 > PAYCO인증서

    1. 위에 표시된 보안문자를 입력해 주세요.

    2. 인증하시는 분의 간편인증 업체를 선택한 후, 서비스 이용 약관에 동의하고 인증요청 버튼을 클릭하세요.

    3. 입력한 휴대폰 번호로 ‘정부24에서 인증요청이 왔습니다‘ PUSH 메시지가 발송됩니다. 간편인증 업체에 따라 비밀번호, 패턴, 생체 인식 등을 통해 본인인증을 완료한 후, ‘인증완료’ 버튼을 클릭해주세요.

    4. 정부24에서 주민등록등본 자동 조회를 시작합니다.

    5. 경기도의 해당 시·군의 거주확인이 완료되었습니다.

    공동·금융 인증 방법
    [공동 인증 방법]

    1. 위에 표시된 보안문자를 입력해 주세요.

    2. 인증하는 분의 공동인증서를 선택해 주세요.

    3. 인증서의 암호를 입력해 주세요.

    4. 정부 24에서 주민등록등본 자동 조회를 시작합니다.

    5. 경기도의 해당 시·군의 거주확인이 완료되었습니다.

    [금융 인증 방법]

    ※ 금융 인증은 모바일에서만 가능합니다.

    1. '정보 조회하기'를 터치하고, 인증서 제출을 '금융인증서'로 선택해주세요.

    2. 위에 표시된 보안문자를 입력하고, '다음'을 터치하세요.

    3. 인증하시는 분의 이름, 휴대폰번호, 생년월일을 입력한 후 확인 버튼을 터치하세요.

    4. 입력한 휴대폰 번호로 전송된 문자를 확인하시고, 확인코드 숫자를 입력해주세요.

    5. 인증하시는 분의 금융인증서를 선택해주세요.

    6. 금융인증서 비밀번호 6자리를 입력해주세요.

    7. 경기도의 해당 시·군의 거주확인이 완료되었습니다.

    인증이 완료된 후 다음 단계로 이동합니다.

    가입을 원하시는 아이디로 입력하고, 중복확인을 해주세요.

    영문자와 숫자, 특수문자 1개 이상을 사용한 비밀번호를 입력해 주세요.

    자녀가 추가된 경우 등록된 자녀의 정보가 자동으로 입력 되어집니다.
    (※ 마이페이지 > 정보수정에서 ‘자녀추가’를 별도로 진행할 수 있습니다.)

    모든 정보 입력이 완료된 후 가입을 완료해 주세요.

    교통카드 등록 팝업에서 교통카드를 등록해주세요.

    교통카드 등록 방법
    [선불 교통카드]

    1. 자녀, 교통카드 종류, 카드사를 선택한 후 교통카드 번호를 입력해주세요.
    (※ 카드사 별로 교통카드 번호 확인 방법이 다르니, ‘번호확인 안내‘ 팝업 내용을 확인해주세요.)

    2. 자녀의 카드가 등록되면, 지역화폐를 등록해주세요.

    [후불 교통카드]

    1. 자녀, 교통카드 종류, 카드사를 선택한 후 교통카드 번호를 입력해주세요.
    (※ 카드사 별로 교통카드 번호 확인 방법이 다르니, ‘번호확인 안내‘ 팝업 내용을 확인해주세요.)

    2. '교통카드 등록'을 클릭 후, 카드 인증을 완료해주세요. (※ 카카오뱅크, 씨티카드, 가족카드는 인증 절차가 없습니다.)

    3. 자녀의 교통카드가 등록되면, 지역화폐를 등록해주세요

    지역화폐 등록 팝업에서 지역화폐를 등록해주세요.

    지역화폐 등록 방법
    [선택한 자녀의 지역화폐 등록]

    1. 휴대폰으로 인증 번호를 발송하고, 인증번호 6자리를 입력 후 인증을 완료해주세요.

    2. 선택한 자녀의 지역화폐 번호를 입력해주세요.
    (※ 성남시, 시흥시, 김포시는 지역화폐 번호 입력을 하지 않습니다.)

    3. 지역화폐를 등록 후, 지원금을 신청해주세요.

    [본인 명의가 아닌 지역화폐 등록]

    1. 대리인의 휴대폰으로 인증 번호를 발송하고, 인증번호 6자리를 입력 후 인증을 완료해주세요.

    2. 대리인의 지역화폐 번호를 입력해주세요.
    (※ 성남시, 시흥시, 김포시는 지역화폐 번호 입력을 하지 않습니다.)

    3. 지역화폐를 등록 후, 지원금을 신청해주세요.

    교통카드와 지역화폐를 등록 후, 지원금 신청하기 버튼을 클릭해주세요.

    지원금 신청을 위해 지원금 신청 페이지로 이동합니다.
    (※ 다음에 신청하시는 경우, 마이페이지 > 지원금 신청에서 지원금을 신청하실 수 있습니다.)

    대리인의 인증서/휴대폰 인증이 모두 가능한 경우
