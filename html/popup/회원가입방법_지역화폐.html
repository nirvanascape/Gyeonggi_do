지역화폐 정보 및 전화번호 변경 가이드

    • 1. 지역화폐 등록정보 변경방법
      • ※ 휴대폰 번호가 변경되신 경우 1,2절차를 모두 진행해 주셔야 합니다.

        경기도 청소년 교통비 지원 포털 (https://www.gbuspb.kr) 에서 로그인 메뉴 클릭

        아이디 및 비밀번호 입력 후 로그인버튼 클릭

        로그인 후 상단 마이페이지 메뉴 클릭

        마이페이지 화면의 지역화폐 등록 메뉴 클릭 후 지역화폐 등록 화면으로 이동

        지역화폐 등록/변경 버튼 클릭

        지역화폐 등록 팝업화면에서 본인 지역화폐를 등록할 경우에는 “본인지역화폐 등록“, 본인명의 아닌 경우 “본인 명의가 아닌 지역화폐 등록” 을 선택

        7-1 본인 지역화폐 정보 입력 후 저장

        7-2 대리인의 지역화폐 정보를 입력 후 저장

    • 2. 홈페이지 내 회원정보 변경방법
      • ※ 휴대폰 번호가 변경되신 경우 1,2절차를 모두 진행해 주셔야 합니다.

        마이페이지 에서 “정보수정”을 선택

        정보 수정 화면에서 휴대폰 번호를 수정

        저장 버튼 클릭하여 정보를 변경처리
